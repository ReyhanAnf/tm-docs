# Glossary & Terminology

This document serves as the central dictionary for all technical and business terms used within the Talents Mapping platform (2026 Transformation).

## Core Concepts

| Term | Definition |
| --- | --- |
| **SKU / Varian** | The smallest unit of a product variant that has a unique code and specific price. |
| **Entitlement (*Tiket Digital*)** | A user's digital right to access a service, decoupled from financial transaction data. It represents "ownership" of a product/service. |
| **Commission Recipient** | Technical term for individuals (Influencers/Practitioners/Affiliates) eligible to receive commissions. |
| **Commission Tier** | The level of a Commission Recipient (e.g., *Influencer*, *Practitioner*, *General Affiliate*) which determines their revenue share percentage. |
| **Service License** | A special type of *entitlement* specifically for service-based offerings (e.g., Consultations). |
| **Hard-Bundling** | A sales package where a single price covers multiple products (e.g., Assessment + Consultation) that cannot be separated or purchased individually within that package. Internally treated as a collection of products. |
| **Delta Price** | A pricing method for upgrades where the user pays only the difference between the new product price and the old product price. |
| **Deep Copy** | The technical process of copying old assessment data to a new ticket intact during an upgrade, ensuring the user does not need to redo the assessment. |
