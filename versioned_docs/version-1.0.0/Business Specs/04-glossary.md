# Glossary & Terminology

Daftar istilah dan definisi yang digunakan dalam dokumentasi serta kode Talents Mapping.

## Roles & Actors

| Term | Definisi |
| :--- | :--- |
| **Member** | Pengguna perorangan (Retail) yang membeli produk untuk diri sendiri. |
| **Partner** | Pengguna B2B (Sekolah, Korporasi, Reseller) yang mendistribusikan asesmen secara massal. |
| **Partner Operator** | Sub-akun dari Partner utama yang hanya memiliki akses untuk mengelola klien tertentu. |
| **Practitioner** | Konselor atau Praktisi Talents Mapping yang berwenang melakukan verifikasi (Feedback Session). |
| **Admin** | Pengelola sistem Back-office (Superadmin). |
| **Client** | Peserta tes yang diundang oleh Partner (bukan member retail). |

## Products & Features

| Term | Definisi |
| :--- | :--- |
| **ST-30** | Strength Typology 30. Salah satu jenis asesmen bakat (versi singkat/lite). |
| **TMA** | Talents Mapping Assessment. Asesmen lengkap yang mencakup 34 bakat. |
| **PSS** | Personal Strength Statement. Bagian dari asesmen untuk mengukur pernyataan kekuatan diri. |
| **Grit** | Skala ketabahan/kegigihan. Bagian dari asesmen. |
| **Quota** | Sistem limitasi untuk Partner tipe `Pre-paid`. 1 Quota = 1 Peserta Laporan. |
| **Reference Code** | Kode unik yang di-generate oleh Partner untuk dibagikan ke peserta tes. |
| **Feedback Session** | Sesi konsultasi tatap muka/online antara User dan Practitioner untuk membedah hasil asesmen. |

## Technical Terms

| Term | Definisi |
| :--- | :--- |
| **Xendit** | Payment Gateway utama untuk memproses pembayaran otomatis. |
| **Temabakat API** | Layanan eksternal (Legacy System) yang digunakan untuk men-generate file PDF report. |
| **Waha** | Layanan Gateway WhatsApp untuk pengiriman notifikasi otomatis. |
| **Snap Token** | Token sesi pembayaran dari Midtrans/Xendit untuk frontend checkout. |
